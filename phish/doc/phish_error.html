phish_error() function

phish_warn() function

C syntax: 

void phish_error(char *str) 

void phish_warn(char *str)

C examples:

phish_error("Bad datum received"); phish_warn("May overflow internal buffer");

Python syntax: 

void phish_error(char *str) 

void phish_warn(char *str)

Python examples:

phish_error("Bad datum received"); phish_warn("May overflow internal buffer");

Description:

There are PHISH library functions which can be called from a minnow application. In PHISH lingo, a "minnow" is a stand-alone application which makes calls to the PHISH library.

The phish_error() function prints the specified character string (str) to the screen, and caused the running minnow to exit.

does not close output ports for MPI, calls MPI_Abort() Do I print additional info about minnow?

The phish_warning() function simply prints the specified character string (str) to the screen. Control is returned to the caller (the minnow) which continues on.
