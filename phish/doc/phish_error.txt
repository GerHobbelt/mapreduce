[Description:]

There are PHISH library functions which can be called from a minnow
application.  In PHISH lingo, a "minnow" is a stand-alone application
which makes calls to the "PHISH library"_Library.html.

The phish_error() function prints the specified character string (str)
to the screen, and caused the running minnow to exit.

does not close output ports
for MPI, calls MPI_Abort()
Do I print additional info about minnow?

The phish_warning() function simply prints the specified character
string (str) to the screen.  Control is returned to the caller (the
minnow) which continues on.
