Papers using the MapReduce-MPI library

We're working on a couple of papers using MapReduce algorithms for informatics problems, which will use the MR-MPI library. They'll be posted here when available. Until then, please cite the MR-MPI WWW page if you use MR-MPI in a paper.

The authors of the library are Steve Plimpton (sjplimp at sandia.gov) and Karen Devine (kddevin at sandia.gov).

  • If you use MR-MPI for something interesting, we'd like to hear about it.
  • If you write a MapReduce program which calls the library, that others would be interested in, you can send it and we'll include it in the distribution.
  • If you enhance the library itself or have ideas for doing so, we can discuss including the improvements in future releases.

You can send us email regarding any of these issues.
